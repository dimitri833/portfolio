Game Over

Voor deze typografie poster had ik in gedachte om aan de hand van een woord een object te maken. Dus de woorden op zo'n manier opstellen waardoor het lijkt alsof er in dit geval een arcade machine staat. Voor het geval dat nog niet duidelijk genoeg was heb ik het woord arcade gebruikt. Ik was eerst van plan om de zijkant van zo'n machine te maken. Dit was alleen te makkelijk en vond ik onorigineel. Dus ik dacht laat ik een beetje met perspectief werken en zo de voorkant van de machine maken. Op deze manier heeft de arcade machine ook diepte gekregen. Verder is die heel simplistisch. Het lettertype waarvoor ik ben gegaan vond ik erg veel weghebben van de gaming stijl. Voornamelijk omdat deze zo pixelated was. Ik heb feedback voor deze poster gevraagd tijdens de wekelijkse design feedback ronde op donderdag en de poster was goedgekeurd. Ook vond de desbetreffende docent het wel leuk gedaan dat de woorden samen een arcade machine vormden. Zelf vind ik het mijn beste typografie poster.
