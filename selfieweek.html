DIMITROVISCH

Dimitrovisch is een artiestennaam die ik daadwerkelijk gebruik. Hiervoor maak ik meestal de genres chillwave, chillsynth, synthwave en vaporwave. Ik heb hier ook al wel wat dingen mee bereikt qua views waar ik dus best trots op ben. Het is wat meer rustgevendere muziek.

FREE4ALL

Dit is een artiestennaam die ik voor dit project heb verzonnen. Hiervoor wil ik ook alleen verschillende vormen van house maken. Ik heb al wat projecten klaarliggen alleen wil ik echt alles afhebben voordat ik deze publiekelijk maak.

TERROR RAIN

Deze naam heb ik verzonnen voor wat heftigere muziek, bijvoorbeeld: techno, hardstyle en hardcore. Hiermee wil ik een wat ruwere kant van mij laten overkomen.
